# IVANNA-OMEGA-SUPREME

**Procesador de audio neuronal para Android** con IA, baja latencia y espacialización 3D.

## 📊 Características reales
- **YAMNet** clasifica 521 sonidos en tiempo real.
- **PiLSTM** mejora la claridad y separación de fuentes.
- **Upmixer IA** convierte estéreo a 5.1 (modelo TFLite).
- **AAudio** con buffer de 128 muestras → latencia < 5 ms.
- **Phase Oracle** corrige la fase en 10 bandas.
- **Control por voz** (comandos: "subir volumen", "modo cine", etc.).
- **Visualizador GLSL** en tiempo real.
- **Anti-Dolby** con curva inversa para EAC3.
